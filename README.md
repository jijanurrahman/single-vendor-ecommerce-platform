# Single Vendor E-commerce Website

> A single-vendor e-commerce platform with modern user authentication (Google & Facebook), product browsing, cart management, and secure checkout via SSLCommerz Payment Gateway. Designed for smooth shopping experience and easy integration for small business online stores.
